# 让项目管理也AI-Native-两个Git仓库干掉了周报

## 一句话概括

用两个 Git 仓库加 AI 编码助手和 Shell 脚本替代传统项目管理中约 80% 的人工操作，核心理念是"降低填写门槛、让 AI 做信息搬运工、人只做输入和决策"。

## 摘录

> 第一个仓库负责业务进展跟踪，采用三层架构（原始 Markdown 自由输入 → AI 语义整合为结构化文档 → 生成静态 Dashboard 可视化），配备问题挖掘 Skill 自动识别跨团队阻塞和基建依赖热点。

> 第二个仓库负责效能度量，通过大数据平台 CLI 拉取数据生成交互式报表。核心理念是"降低填写门槛、让 AI 做信息搬运工、人只做输入和决策"。
